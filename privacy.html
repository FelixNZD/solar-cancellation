Privacy Policy

Last updated: February 27, 2026  ·  Effective immediately

Summary: This website does not collect, store, or sell your personal data. We use no cookies, no tracking pixels, and no third-party analytics. When you click "Yes," you leave this site and the destination's own privacy policy applies.

1. Information We Collect

This website is a single-page informational advertisement. We do not operate contact forms, account registration, or any data submission features. We do not directly collect, store, or process any personally identifiable information (PII).

Our web server may automatically log limited, non-personal technical data as part of standard server operations:

  • IP address (not stored or linked to individuals)
  • Browser type and version
  • Operating system
  • Referring page URL
  • Date and time of page request
  • HTTP response codes

This technical data is used exclusively for server security, diagnosing errors, and ensuring page availability. It is not shared, sold, or linked to any individual identity.

2. How We Use Information

The limited technical data described above is used solely for the following purposes:

  • Maintaining the availability and performance of this website
  • Identifying and resolving technical errors
  • Protecting against unauthorized access or abuse

We do not use any collected data for advertising targeting, profiling, remarketing, or any commercial purpose beyond operating this page.

Cookies & Tracking None used

This website does not use cookies of any kind — not session cookies, persistent cookies, or third-party cookies. We do not use:

  • Tracking pixels or web beacons
  • Browser fingerprinting scripts
  • Local storage or IndexedDB
  • Meta (Facebook) Pixel
  • Google Analytics or Google Tag Manager
  • TikTok, Snapchat, or any other ad platform pixels
  • Any third-party JavaScript libraries or SDKs

All code on this page is self-contained. There are zero external network requests made when this page loads.
